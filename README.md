Fitness-Quantified
==================

Passively track exercise using sensors attached to fitness equipment.



Proposed Devices
-------------

1. Pull Ups
2. Dumbbells
3. Sit Ups
4. Push Ups
5. Jump Rope
6. Laziness (sitting, tv)
